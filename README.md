# 🍽️ M.S Da Dhabba

A simple and responsive food recipe website built using HTML and CSS.  
It showcases popular Indian desserts like Gulab Jamun and Barfi with ingredients and step-by-step recipes.

## 🚀 Features
- Clean UI design
- Flexbox-based layout
- Image hover effects
- Structured recipe sections

## 💻 Tech Stack
- HTML
- CSS
